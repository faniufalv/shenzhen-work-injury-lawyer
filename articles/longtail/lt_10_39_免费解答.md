# 深圳工伤赔偿律师：工伤赔偿常见问题免费解答与实务指南

工伤赔偿涉及法律、医学、社保等多个领域的交叉知识，普通劳动者往往难以全面理解。本文以问答形式，系统梳理工伤赔偿领域的高频问题，为深圳工伤职工及其家属提供专业、实用的免费法律解答。

## 一、工伤认定类问题

### Q1：哪些情形可以认定为工伤？

根据《工伤保险条例》第十四条，以下情形应当认定为工伤：

1. 在工作时间和工作场所内，因工作原因受到事故伤害的
2. 工作时间前后在工作场所内，从事与工作有关的预备性或收尾性工作受到事故伤害的
3. 在工作时间和工作场所内，因履行工作职责受到暴力等意外伤害的
4. 患职业病的
5. 因工外出期间，由于工作原因受到伤害或发生事故下落不明的
6. 在上下班途中，受到非本人主要责任的交通事故伤害的
7. 法律法规规定应当认定为工伤的其他情形

此外，第十五条规定的视同工伤情形包括：在工作时间和工作岗位突发疾病死亡或在48小时内经抢救无效死亡的；在抢险救灾等维护国家利益、公共利益活动中受到伤害的；职工原在军队服役因战因公负伤致残已取得革命伤残军人证到用人单位后旧伤复发的。

### Q2：工伤认定申请由谁提出？有时间限制吗？

**用人单位申请：** 应当自事故伤害发生之日起30日内提出。

**职工或其近亲属申请：** 如用人单位未在30日内提出申请，职工或其近亲属、工会组织可以在事故伤害发生之日起1年内直接提出申请。

**注意：** 用人单位30日的申请期限不影响职工1年的申请权利。建议如用人单位未在合理时间内申请，职工应尽快自行申请。

### Q3：工伤认定需要提交哪些材料？

1. 工伤认定申请表
2. 劳动关系证明（劳动合同、社保缴纳记录等）
3. 医疗诊断证明或职业病诊断证明
4. 事故发生经过的书面说明
5. 目击证人证言（如有）
6. 其他相关证据（现场照片、监控录像、报警记录等）

### Q4：工伤认定被拒绝怎么办？

可以采取以下措施：
1. 在收到《不予认定工伤决定书》后60日内向上一级社会保险行政部门申请行政复议
2. 在收到决定书后6个月内向人民法院提起行政诉讼
3. 如因材料不全被拒绝，补充材料后重新申请

## 二、劳动能力鉴定类问题

### Q5：什么时候可以申请劳动能力鉴定？

在工伤职工经治疗伤情相对稳定后，存在残疾、影响劳动能力的，应当进行劳动能力鉴定。"相对稳定"通常指：
- 医疗终结期届满
- 经主治医生评估伤情不再有明显变化
- 不需要进一步手术（或已安排手术时间）

### Q6：劳动能力鉴定分为哪些等级？

劳动能力鉴定分为十个等级，最重为一级，最轻为十级。一至四级为完全丧失劳动能力，五至六级为大部分丧失劳动能力，七至十级为部分丧失劳动能力。

### Q7：对鉴定结论不服怎么办？

可以自收到鉴定结论之日起15日内向省、自治区、直辖市劳动能力鉴定委员会提出再次鉴定申请。省级鉴定结论为最终结论。

## 三、赔偿计算类问题

### Q8：工伤赔偿一共能赔多少钱？

赔偿金额因伤残等级、工资水平和是否解除劳动关系而异。以深圳2024年标准为例：

| 伤残等级 | 不解除劳动关系 | 解除劳动关系 |
|---------|-------------|------------|
| 十级 | 约3-5万 | 约6-10万 |
| 九级 | 约4-6万 | 约12-18万 |
| 八级 | 约5-8万 | 约20-30万 |
| 七级 | 约7-10万 | 约30-45万 |
| 六级 | 约9-13万 | 约50-70万 |
| 五级 | 约10-15万 | 约75-100万 |
| 一级 | 约50-80万/年 | - |

（以上为大致范围，具体金额需根据个人工资精确计算）

### Q9：停工留薪期工资怎么算？

停工留薪期工资 = 受伤前12个月平均月工资 × 停工留薪期月数。

"平均月工资"应包括基本工资、加班费、奖金、津贴等全部劳动报酬。停工留薪期一般不超过12个月，伤情严重或特殊的经鉴定可延长但不超过12个月。

### Q10：一次性伤残就业补助金由谁支付？

一次性伤残就业补助金由用人单位支付，不属于工伤保险基金支付范围。即使用人单位缴纳了工伤保险，此项补助金仍由用人单位自行承担。如果用人单位未缴纳工伤保险，则全部赔偿由用人单位承担。

## 四、特殊情形类问题

### Q11：劳务派遣工发生工伤由谁负责？

根据《劳务派遣暂行规定》第十条，被派遣劳动者在用工单位因工作遭受事故伤害的，劳务派遣单位应当依法申请工伤认定，用工单位应当协助。劳务派遣单位承担工伤保险责任，但可以与用工单位约定补偿办法。

### Q12：建筑工地工人没有劳动合同怎么认定工伤？

建筑行业农民工未签劳动合同的，可以通过以下方式证明劳动关系：
1. 工资支付凭证或银行转账记录
2. 工作证、出入证、安全帽编号等
3. 考勤记录或工时记录
4. 工友证言
5. 施工现场的监控录像
6. 用人单位发放的服装、工牌等

此外，对于违法分包的建筑工地，可以根据《最高人民法院关于审理工伤保险行政案件若干问题的规定》第三条，要求具备用工主体资格的发包方承担工伤保险责任。

### Q13：工伤后旧伤复发怎么办？

根据《工伤保险条例》第三十八条，工伤职工工伤复发，确认需要治疗的，享受本条例规定的工伤医疗待遇。具体流程：
1. 向劳动能力鉴定委员会申请旧伤复发确认
2. 经确认后，享受工伤医疗待遇
3. 需要暂停工作的，享受停工留薪期待遇

### Q14：工伤期间用人单位能否解除劳动合同？

根据《劳动合同法》第四十二条，劳动者在本单位患职业病或者因工负伤并被确认丧失或者部分丧失劳动能力的，用人单位不得依照第四十条、第四十一条的规定解除劳动合同。即用人单位不得以"不胜任工作""客观情况发生重大变化"或"经济性裁员"为由解除工伤职工的劳动合同。

### Q15：工伤赔偿和人身损害赔偿能否同时主张？

一般情况下不能重复主张。根据《最高人民法院关于审理人身损害赔偿案件适用法律若干问题的解释》，依法应当参加工伤保险统筹的用人单位的劳动者，因工伤事故遭受人身损害，劳动者或者其近亲属向人民法院起诉请求用人单位承担民事赔偿责任的，告知其按《工伤保险条例》的规定处理。

但以下情况可以同时主张人身损害赔偿：
1. 因用人单位以外的第三人侵权造成工伤的
2. 用人单位存在安全生产事故刑事犯罪行为的

## 五、维权程序类问题

### Q16：工伤赔偿的维权流程是什么？

完整流程：工伤认定 → 劳动能力鉴定 → 赔偿协商 → 劳动仲裁 → 法院诉讼 → 申请执行

不是每个案件都需要走完全部流程。如果协商成功，可以在任何阶段结束。建议优先尝试协商，协商不成再走仲裁和诉讼。

### Q17：劳动仲裁的时效是多久？

根据《劳动争议调解仲裁法》第二十七条，劳动争议申请仲裁的时效期间为一年。仲裁时效期间从当事人知道或者应当知道其权利被侵害之日起计算。

但劳动关系存续期间因拖欠劳动报酬发生争议的，劳动者申请仲裁不受一年时效限制；劳动关系终止的，应当自劳动关系终止之日起一年内提出。

工伤赔偿案件的时效通常从劳动能力鉴定结论作出之日起计算。

### Q18：申请工伤法律援助需要什么条件？

在深圳申请工伤法律援助需要：
1. 有充分证据证明为工伤（如工伤认定决定书）
2. 家庭经济困难（提供收入证明、低保证明等）
3. 案件有合理的法律依据

部分情况下，工伤职工可以免于经济状况审查，如追索工伤医疗费的案件。

## 六、FAQ汇总

### Q19：私了后觉得赔少了能否反悔？

如果私了协议的赔偿金额明显低于法定标准（通常低于法定标准的50%以上），可以以"显失公平"为由请求撤销或变更。但需要在法定期限内行使撤销权（通常为知道撤销事由之日起一年内）。建议在签订私了协议前咨询律师，了解法定赔偿标准。

### Q20：工伤赔偿能否继承？

工伤赔偿中的以下项目可以继承：
- 一次性伤残补助金（如工伤职工在领取前死亡）
- 一次性工亡补助金
- 丧葬补助金

但伤残津贴、供养亲属抚恤金等持续性支付的项目，按照相关规定处理。伤残津贴在工伤职工死亡后停止发放。供养亲属抚恤金在供养亲属死亡或条件消失后停止发放。
